== Description ==

Download and Insert images to WP Media Library from External URLs.

== Installation ==

Install from wordpress.org:

01. Go to Plugins > Add New.
02. Search for "GL Import External Images"
03. Install and Active it. No configuration need.

Install from ZIP file:

01. Upload the plugin files to the `/wp-content/plugins/` directory.
02. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= I can download any format? =

WordPress Default formats ( 'jpg', 'jpeg', 'jpe', 'png', 'gif', 'webp' ) are supported.

= What will be image name? =

Image name will be the file name from your URL.

= Can I import from all URL? =

Your given URL must be publicly accessible.
